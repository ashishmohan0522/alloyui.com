Viewport - Basic Example

Adjust your browser's width and see how the page adapts!

Watch me change! 
AUI().use('aui-viewport');

.aui-view-lt960 #example {
  background: #777;
  color: #FFF;
  width: 450px;
}

.aui-view-lt720 #example {
  font-size: 32px;
  text-shadow: -1px -1px 0 #00F, 1px -1px 0 #00F, -1px 1px 0 #00F, 1px 1px 0 #00F;
  width: 200px;
}
